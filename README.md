vidmod
======

```
vidmod 1.1.0

USAGE: vidmod [<options>] <change1> [<changen>...] <input>

Do rapid changes to video files using ffmpeg or other tools.

CHANGES:

	169, 43, 60fps, audio, butter, crop219, crossfade, loop, mono, mp4, mute,
	qt, reverse, rotate90, rotate180, rotate270, slowdown, twitter

OPTIONS:

	-d    Output directory. Defaults to directory of input file.
	-o    Last output file. When specified, any intermediate file will
	      be temporary.
	-h    Show this help screen.
	-v    Show version.

```
